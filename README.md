# Current Weather and 5-Day Weather Forecast

A simple application to display current weather of user location and 5-day weather forecast using the OpenWeatherMap API.

## Pre-requisites

- Node.js 12 and above & Expo-cli

## Run

```
git clone
cd weather-app
npm install
npm start

```
